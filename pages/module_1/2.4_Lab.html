2.4 Lab: Set up an A/V Workstation

Step 1: Select and Install the Video Card

  • Above the computer, select the change view icon, then select Motherboard to switch to the motherboard view of the computer.
  • Under Inventory, expand GPUs/Expansion Cards.
  • Drag the Video Adapter, DVI-I, HDMI, Crossfire, PCIe to the PCIe slot on the computer.
  • Select the power supply.
  • From the Selected Item pane, drag the Connector, Power Supply, 6-pin PCIe power connector into the video card power slot.
  • Select GPUs/Expansion Cards to return to the main Inventory list.

Step 2: Install Three Hard Drives for the RAID5 Array

  • Under Inventory, expand Storage.
  • Drag all three hard drives to a free 3.5" drive bay.
  • Under Inventory, expand Cables.
  • Select a Cable, SATA, and from the Selected Item pane:
  • Drag an unconnected SATA connector to a hard drive.
  • Drag the other unconnected SATA connector to an open SATA port on the motherboard.
  • Repeat these steps to connect cables to the other two hard drives.
  • Select the partial Connections icon by the computer, then select the power supply.
  • From the Selected Item pane, drag an unconnected SATA power connector to a hard drive.
  • Repeat the process for the remaining drives.
  • Select Cables under Inventory to return to the main list.

Step 3: Connect the Original Monitor to the New Video Card

  • Above the computer, select the change view icon, then select Back.
  • On the computer, drag the DVI connector from the built-in video connector to the new video card in the computer.

Step 4: Connect a Second Monitor

  • Under Inventory, expand Peripherals.
  • Drag the Monitor to the left of the desktop.
  • Above the left monitor, select the change view icon, then select Back.
  • Under Inventory, expand Cables.
  • Select the HDMI to HDMI Cable.
  • From the Selected Item pane:
  • Drag an HDMI Connector to the new video card in the computer.
  • Drag the unconnected HDMI Connector to the HDMI port on the left monitor.
  • Under Inventory, select the AC Power Cable.
  • From the Selected Item pane:
  • Connect the Female AC Power Connector to the left monitor.
  • Connect the Male AC Power Connector to a wall outlet.
  • Above the left monitor, select the change view icon, then select Front.
  • Select the Power button to power on the left monitor.

Step 5: Power On the Computer and Enable RAID in BIOS

  • Above the computer, select the Change View icon, then select Front.
  • Select the power button to turn on the computer.
  • As the computer boots, press F2 (or Delete) to enter the BIOS setup program.
  • From the left menu, expand System Configuration.
  • Select SATA Operation.
  • From the right pane, select RAID On.
  • Select Apply.

Step 6: Exit BIOS and Configure the RAID Array

  • Select Exit to restart the system.
  • After the system boots and the BIOS loads, the RAID controller will load.
  • When the RAID message screen displays, press Ctrl + I (i).
  • With Create RAID Volume highlighted, press Enter.
  • On the Name field, press Enter.
  • On the RAID Level field, use the up and down arrow keys to select RAID5 (Parity).
  • Press Enter.
  • With Select Disks highlighted, press Enter.
  • Use the up and down arrow keys to traverse the list and the spacebar to select a drive.
  • Select all three drives with the model number WD800JD and press Enter.
  • Press the Tab key.
  • Verify that the Strip size is 128 KB. If needed, use the up and down arrows to set the value.
  • Press Enter.
  • Press Enter to accept the default capacity.
  • With Create Volume highlighted, press Enter.
  • Press Y to continue creating the volume.
  • Press the escape key (Esc).
  • Press Y to exit.
