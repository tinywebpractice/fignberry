3.2.13 Lesson Review

Q1 - Which Blu-ray standard defines rewritable discs?

Correct answer: BD-RE

BD-RE stands for Blu-ray Disc Recordable Erasable - the rewritable Blu-ray format. BD-ROM is read-only, and BD-R is write-once.

Q2 - Which BEST describe SSDs compared to HDDs? (Select two.)

Correct answers: An SSD uses flash technology to store data.
An SSD opens files faster than an HDD.

SSDs have no moving parts and use flash memory, making them faster than HDDs. They are not affected by magnetism and have no spindles. They are also more expensive per GB than HDDs.

Q3 - Which storage device uses aluminum platters for storing data?

Correct answer: Hard disk

HDDs store data magnetically on metal (aluminum or glass) platters. SSDs, SD cards, and optical discs do not use platters.

Q4 - Which drive configuration uses striping with parity for fault tolerance?

Correct answer: RAID 5

RAID 5 uses striping with distributed parity, allowing recovery from a single disk failure. RAID 0 = striping only (no fault tolerance). RAID 1 = mirroring. RAID 10 = stripe of mirrors.

Q5 - Which RAID method should you use? (Hardware RAID controller card scenario)

Correct answer: Hardware

When a dedicated RAID controller card is present, hardware RAID is the correct method. It runs before the OS boots and offers more control and options than software RAID.

Q6 - Which storage media BEST allows quick, inexpensive file sharing with customers?

Correct answer: Flash drive

Flash drives (USB drives) are portable, inexpensive, and plug into any USB port - ideal for quickly sharing files. HDDs and SSDs are bulkier and more expensive; SDHC cards require a card reader.

Q7 - What distinguishes SD Express cards from earlier SD versions?

Correct answer: They offer much higher speeds

SD Express combines the SD card form factor with PCIe and NVMe interfaces, achieving speeds up to 985 MBps - far beyond earlier standards like UHS (108 MBps).

Q8 - Which BEST meets your needs? (Two drives, increased performance)

Correct answer: RAID 0

RAID 0 (striping) splits data across two drives to improve performance. It requires a minimum of 2 drives and uses all available capacity. Note: there is no fault tolerance with RAID 0.

Q9 - Windows doesn't show the RAID 1 logical drive after configuring with the motherboard controller. What should you do?

Correct answer: Install the driver for the RAID controller.

When using an integrated motherboard RAID controller, Windows requires a driver to recognise the RAID volume. Without it, the logical drive won't appear in the OS.

Q10 - Which RAID BEST meets: fault tolerance, minimum drives, low cost?

Correct answer: RAID 1

RAID 1 requires only 2 drives, provides fault tolerance through mirroring, and is the simplest and least expensive redundancy option. RAID 5 and RAID 10 require more drives.

Q11 - What is an advantage of RAID 5 over RAID 1?

Correct answer: RAID 5 improves performance over RAID 1.

RAID 5 uses striping alongside parity, giving it better read performance than RAID 1. Both provide fault tolerance, but RAID 5 tolerates only 1 disk failure (same as RAID 1) and uses disk space more efficiently.

Q12 - Data storage recommendations for a high performance system? (Select two.)

Correct answers: Store heavily used information on the hard drive.
Store data that requires high performance on the SSD drive.

Use the hard drive for general storage needs and put performance-sensitive data on the SSD. This balances capacity and cost while still using the SSD where speed matters most.

Q13 - What do you need to do with Disk 1 and Disk 2 before configuring RAID 1 in Windows?

Correct answer: Convert the disks from basic to dynamic.

Windows software RAID requires dynamic disks. Basic disks must be converted to dynamic before they can be used in a mirrored (RAID 1) or striped (RAID 0) volume.

Q14 - Best solution for two drives, protection from disk failure, Windows software RAID?

Correct answer: Use mirrored volumes.

A mirrored volume (RAID 1) in Windows Disk Management duplicates data across two drives. If one fails, the other takes over. RAID 5 requires at least 3 drives and is not available as a software RAID option in standard Windows.

Q15 - A RAID 1 array with two 1 TB drives only shows 1 TB of available space. What should you do?

Correct answer: Nothing. The RAID array is working as designed.

RAID 1 (mirroring) uses 50% of total disk space. With two 1 TB drives, only 1 TB is usable - the second drive is a hidden mirror copy. This is expected behaviour.
