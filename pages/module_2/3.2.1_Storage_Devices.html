3.2.1 Storage Devices

Volatile vs Non-Volatile Memory

Type Retains Data Without Power? Purpose Example
Volatile No — data is lost when power is removed Short-term; moves data quickly in and out RAM
Non-volatile Yes — data is retained even when powered off Long-term; stores programs, files, images, music Hard drives, SSDs

Magnetic Storage Devices (HDD)

  • Store data magnetically on flat metal plates called platters
  • Less expensive and available in very large capacities
  • Slower data transfer due to moving parts
  • Moving parts can physically break, causing data loss
  • Available in internal and external models

Internal hard drive sizes:

Size Used In
3.5 inch Desktop computers
2.5 inch Laptops and small form factor PCs

Note: Internal hard drives connect to the motherboard using a SATA (Serial ATA) cable.

Solid-State Drives (SSD)

  • Store data using flash memory modules — no moving parts
  • Much faster data transfer than magnetic drives
  • More reliable due to no moving parts
  • More expensive than magnetic drives
  • Currently available in smaller maximum capacities than HDDs
  • Data can be written and rewritten many times without power to retain it

Note: Portable flash memory devices (e.g. SD cards, USB drives) are popular for digital cameras, tablets, and smartphones.

Magnetic vs Solid-State Comparison

Feature HDD (Magnetic) SSD (Solid-State)
Speed Slower Much faster
Moving parts Yes (platters) No
Reliability Lower — parts can break Higher
Cost Less expensive More expensive
Capacity Very large Smaller (for now)

External Storage

  • External hard drives – Available as magnetic or flash-based; capacities up to terabytes; connect via USB; ideal for backups
  • NAS (Network-Attached Storage) – Contains multiple drives connected to a network, allowing multiple users to save and retrieve data

CD-ROM and Cloud Storage

  • CD-ROM – Optical disc originally used for music, later for programs, games, and movies; now largely replaced
  • Cloud storage – The modern long-term storage choice; data is stored remotely and accessed via streaming or download services

Key Terms

  • Volatile memory – Memory that loses data when power is removed (e.g. RAM)
  • Non-volatile memory – Memory that retains data without power (e.g. HDD, SSD)
  • Platter – The flat metal disc inside an HDD where data is stored magnetically
  • SATA (Serial ATA) – The cable/interface used to connect internal storage drives to the motherboard
  • Flash memory – Non-volatile memory used in SSDs and portable storage devices
  • NAS (Network-Attached Storage) – A networked storage device accessible by multiple users
  • CD-ROM – Optical disc storage; largely replaced by cloud services
