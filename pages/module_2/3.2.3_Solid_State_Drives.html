3.2.3 Solid-State Drives

What is an SSD?

A solid-state drive (SSD) uses flash memory technology for persistent mass storage. SSDs offer significantly better read performance than HDDs, are less prone to failure from mechanical shock, and have rapidly decreased in cost per gigabyte.

Note: SSDs can sometimes underperform HDDs when handling constant transfers of multi-gigabyte files.

NAND Flash Memory Types

Type Full Name Reliability Cost
SLC Single-Level Cell Highest Most expensive
MLC Multi-Level Cell Lower than SLC Less expensive

Note: Flash memory degrades over many write operations. Wear leveling routines (used by drive firmware and the OS) evenly distribute writes across all blocks to extend the drive's lifespan.

SSD Interface Types

Interface Form Factors Max Speed Notes
SATA 2.5-inch caddy, mSATA 600 MB/s Can bottleneck high-performing SSDs; mSATA plugs directly into a combined data/power port on the motherboard
PCIe / NVMe PCIe expansion card, M.2 Up to 6.7 GB/s+ Uses NVMe interface for much better performance; M.2 slots provide power over the bus — no separate power cable needed
SAS Enterprise drives Very high Used in servers and workstations; faster and more reliable than SATA; point-to-point serial protocol

M.2 SSDs

  • M.2 is a physical form factor — not an interface; M.2 SSDs can use either SATA/AHCI or NVMe
  • NVMe-based M.2 SSDs typically outperform SATA-based M.2 SSDs
  • Smaller and horizontal — suitable for laptops and PC motherboards
  • M.2 size labels indicate dimensions: e.g. M.2 2280 = 22mm wide × 80mm long
  • M.2 key types: B key (SATA), B/M key (2-lane PCIe), M key (4-lane PCIe)
  • Always check motherboard documentation for M.2 compatibility

Safety

SSDs are vulnerable to electrostatic discharge (ESD). Always take anti-ESD precautions when handling and storing SSDs.

Key Terms

  • SSD – Solid-State Drive; uses flash memory for non-volatile storage with no moving parts
  • NAND flash – The type of flash memory used in SSDs (NOT AND logic gate)
  • SLC – Single-Level Cell; most reliable and expensive NAND type
  • Wear leveling – Firmware/OS technique that distributes writes evenly across flash blocks to extend SSD life
  • NVMe – Non-Volatile Memory Express; high-speed interface for PCIe SSDs
  • mSATA – Mini SATA; a compact SSD form factor that plugs directly into a motherboard port
  • M.2 – A compact physical SSD form factor supporting both SATA and NVMe interfaces
  • SAS – Serial Attached SCSI; enterprise-grade interface used in servers and workstations
  • ESD – Electrostatic Discharge; can damage SSDs during handling
