3.3.2 RAM Types

Evolution of RAM Technology

Type What It Introduced
DRAM (Dynamic RAM) Stores bits as electrical charges in capacitors (charged = 1, uncharged = 0); transistors read the capacitor contents
SDRAM (Synchronous DRAM) Synchronised to the system clock so memory operations are timed with CPU instructions
DDR SDRAM (Double Data Rate) Transmits data on both the rising and falling edges of the clock cycle — doubles data transfer rate

How DDR Speed Labels Work (DDR-200 / PC1600 Example)

Specification Value How It's Calculated
Clock speed 100 MHz Internal memory and bus speed
Data rate 200 MT/s 100 MHz × 2 (DDR transfers on both clock edges) = DDR-200
Peak transfer rate 1600 MBps (1.6 GBps) 200 MT/s × 8 bytes per transfer = PC-1600

Note: MT/s = megatransfers per second — how many million data transfers occur per second.

DDR Generations Comparison

RAM Type Data Rate Transfer Rate Max Module Size
DDR1 200–400 MT/s 1.6–3.2 GB/s 1 GB
DDR2 400–1066 MT/s 3.2–8.5 GB/s 4 GB
DDR3 800–2133 MT/s 6.4–17.1 GB/s 16 GB
DDR4 1600–3200 MT/s 12.8–25.6 GB/s 32 GB
DDR5 4800–8000+ MT/s 38.4–51.2+ GB/s 128 GB or higher

Note: Each DDR generation increases bandwidth by multiplying the bus speed, not the memory device speed — keeping modules reliable and cool.

CAS Latency and Timing

Memory modules have internal timing values expressed as e.g. 14-15-15-35 CAS 14.

  • CAS latency (Column Access Strobe latency) – The delay in clock cycles between the memory controller requesting data and it becoming available
  • Lower values = better performance when comparing modules of the same type and speed
  • CAS latency is a key factor in RAM speed and efficiency

Architectural Improvements

  • DDR4 – Increased power efficiency, higher density (larger capacities), more efficient channel design for better data throughput
  • DDR5 – Further power efficiency improvements; introduces a dual 32-bit subchannel architecture within each DIMM for more efficient access and reduced latency

Key Terms

  • DRAM – Dynamic RAM; stores bits as electrical charges in capacitors
  • SDRAM – Synchronous DRAM; timed to the system clock
  • DDR SDRAM – Double Data Rate SDRAM; transfers data on both clock edges, doubling throughput
  • MT/s – Megatransfers per second; measures how many million data transfers occur per second
  • CAS latency – Column Access Strobe latency; delay (in clock cycles) between requesting and receiving data from RAM; lower = faster
  • PC-xxxx label – Denotes peak transfer rate in MBps (e.g. PC-1600 = 1600 MBps)
  • DDR-xxx label – Denotes data rate in MT/s (e.g. DDR-200 = 200 MT/s)
