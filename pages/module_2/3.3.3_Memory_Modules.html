3.3.3 Memory Modules

What is a Memory Module?

A memory module is a printed circuit board that holds a group of RAM devices acting as a single unit. Each DDR generation sets an upper limit on maximum capacity per module.

Safety: Memory modules are vulnerable to electrostatic discharge (ESD). Always take anti-ESD precautions when handling and storing them.

DIMM vs SODIMM

Feature DIMM SODIMM
Full name Dual Inline Memory Module Small Outline DIMM
Size Larger Smaller
Used in Desktop PCs, workstations, servers, gaming rigs Laptops, mini PCs, compact/small form factor systems
Capacity Higher — better for performance-critical systems Typically 4 GB to 32 GB
Performance Higher Lower — limited by smaller size
Insertion method Press straight down; catches on each end lock it in Insert at a 45° angle, then press flat to lock

Physical Identification

  • Notches on the edge connector identify the DDR generation (DDR3, DDR4, DDR5) and prevent incorrect insertion — each generation's notch is in a different position
  • DIMMs often feature heat sinks to manage heat from high clock speeds
  • Memory slots have catches on each end to secure the module once inserted

Installation and Compatibility

  • The DDR type must match the motherboard — e.g. DDR5 modules cannot be installed in DDR4 slots
  • For best performance, use modules rated at the same bus speed as the motherboard
  • Mixing different speeds is possible but not recommended — the system will run at the speed of the slowest module

Key Terms

  • Memory module – A PCB holding multiple RAM chips that act as a single unit
  • DIMM – Dual Inline Memory Module; standard desktop RAM form factor
  • SODIMM – Small Outline DIMM; compact RAM for laptops and small form factor systems
  • Notch – The keyed slot on a RAM module's edge connector that identifies DDR generation and prevents incorrect installation
  • Heat sink – A metal component on some RAM modules that dissipates heat from high-speed operation
  • ESD (Electrostatic Discharge) – Can damage memory modules; always use anti-ESD precautions when handling RAM
