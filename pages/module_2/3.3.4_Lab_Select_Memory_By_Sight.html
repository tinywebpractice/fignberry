Lab Focus

This lab is about identifying RAM by sight. Pay attention to module size, notch placement, pin layout, and whether the module is a DIMM or SODIMM.

What to Look For

  • Whether the module is a full-size DIMM or a compact SODIMM
  • The position of the notch on the connector edge
  • The overall module shape and connector style
  • Visible differences between DDR generations and laptop versus desktop memory

Important: Memory modules are sensitive to ESD. Handle them by the edges and avoid touching the contacts.
