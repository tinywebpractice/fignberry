3.3.6 Memory Troubleshooting

Primary Troubleshooting Method

The main way to test whether RAM is faulty is to swap it out with a known good stick. If the problem resolves, the original module was the cause.

Identifying the Correct RAM Type

  • The primary way to identify RAM type is by the position of the notch on the module's edge connector
  • Each DDR generation (DDR3, DDR4, DDR5) has its notch in a different position — this prevents incorrect installation
  • A DDR3 stick cannot be inserted into a DDR4 slot — the notch will not line up
  • Always match the module's notch position to the slot before attempting installation

Installing RAM Safely

  • Line up the notch on the module with the key in the slot before pressing down
  • Press firmly but evenly until the retention clips on each end click into place
  • If the RAM does not seat properly, check the notch alignment and orientation before trying again
  • Never force RAM into a slot — forcing it can break the module or damage the motherboard

Troubleshooting Checklist

Problem What to Check / Do
RAM won't seat in slot Verify notch alignment; confirm DDR generation matches the slot; check orientation
System unstable / crashes Swap with a known good stick to test if the module is faulty
Wrong RAM type installed Identify DDR type by notch position; replace with the correct generation
RAM not recognised by system Reseat the module; check it is fully clicked in; verify compatibility with motherboard

Key Rules

  • Do check the notch position to identify DDR type before installing
  • Do use a known good module to isolate a faulty stick
  • Do not mix DDR generations in the same system
  • Do not force RAM into a slot — if it doesn't fit, something is wrong

Key Terms

  • Notch – The keyed slot on a RAM module that identifies its DDR generation and prevents incorrect insertion
  • Known good stick – A verified working RAM module used to test whether a suspected faulty module is the cause of a problem
  • Retention clips – The catches on each end of a DIMM slot that lock the module in place when fully seated
